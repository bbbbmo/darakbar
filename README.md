## 프로젝트 개요

평소 칵테일을 좋아하는데 나만의 칵테일 레시피를 만들고 공유하여 칵테일에 대한 진입장벽을 낮추고자 개발 중입니다.

현재 페이지는 메인 홈 페이지, 유저 칵테일 등록/조회 페이지(나만의 칵테일), 기존 유명한 칵테일 조회 페이지로 구성되어 있습니다.
